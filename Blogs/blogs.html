What is DOM and what is the purpose of DOM?

What is DOM?

The Document Object Model (DOM) is a programming interface for web documents. We can change the document structure, style, and content. Besides we can add or update new content. The DOM represents the document as nodes and objects. In DOM structure HTML file behaves like the document, and all the html tags behaves like object. We can access html tags like the javascript object in DOM.

What is the purpose of DOM?

  1. Locating elements in the document by their ID, class, tag name
  2. Maintain a standard programming interface for HTML and XML documents.
  3. Adding, removing or modifying the HTML elements themselves.
  4. Modifying any attribute of an HTML element, such as their class, style attributes and others
  5. We can even use JavaScript to add animation to a document.

How will you select HTML elements using DOM? Name the DOM methods.

How will you select HTML elements using DOM?

We can select an element based on its unique ID with the getElementById() method. This is the easiest way to find an HTML element in the DOM tree. Similarly, we can use the getElementsByClassName() method to select all the elements having specific class names. This method returns an array-like object of all child elements which have all of the given class names. We can also select HTML elements by tag name using the getElementsByTagName() method. This method also returns an array-like object of all child elements with the given tag name. We can use the querySelectorAll() method to select elements that matches the specified CSS selector. CSS selectors provide a very powerful and efficient way of selecting HTML elements in a document.

Name the DOM methods.

  1. HTML DOM getElementById() Method.
  2. HTML DOM getElementsByClassName() Method.
  3. HTML DOM getElementsByName() Method.
  4. HTML DOM getElementsByTagName() Method.
  5. HTML DOM querySelector() Method.

Actually event bubble means propagate an aevent from the lowest to the highest. It's a concept of DOM. It happens when an element receves an event. And that event bubbles up or tranamitted or propagate to it's parent and ancestor elements (grand parent) in the DOM tree untill it gets the root element.

What is a callback function and why will you use it?

What is a callback function

Any function that is pased as an argument to another function so that it can be exicuted in that other function is called as a callback function. This execution may be immediate as in a synchronous callback, or it might happen at a later point in time as in an asynchronous callback

Why will we use callback function?

JavaScript runs code sequentially in top-down order. However, there are some cases that code runs (or must run) after something else happens and also not sequentially. This is called asynchronous programming. Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors.
